# Air Quality Prediction & Analysis

This project analyzes air quality data across Indian cities and predicts AQI levels using machine learning.

## Features
- Data preprocessing and cleaning
- Exploratory data analysis (EDA)
- AQI prediction using Linear Regression
- City-wise pollutant analysis and visualizations

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Dataset
- Kaggle Air Quality Dataset (29K+ records)
